# Tienda de Electrónicos - Full-Stack Web App

Proyecto universitario desarrollado para simular una plataforma de comercio electrónico dinámica.

## 🛠️ Tecnologías Utilizadas
- **Backend:** PHP
- **Base de Datos:** MySQL
- **Frontend:** HTML5, CSS3, JavaScript

## 🚀 Características
- Autenticación y registro de usuarios de forma segura.
- Carrito de compras funcional con actualización de cantidades.
- Gestión de inventario automatizada conectada a la base de datos.
